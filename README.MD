## 🚀 BingoCode 安装指南

按照以下步骤检查环境并完成安装：

### **第一步：检查环境**
首先，我们需要确认你的电脑是否已经安装了 **npm**。
* 打开终端（CMD 或 PowerShell），输入以下命令：
```bash
npm -v
```
> **结果判断：**
> * ✅ 如果显示版本号（如 `10.x.x`），请跳至 **第三步**。
> * ❌ 如果提示“不是内部或外部命令”，请执行 **第二步**。

---

### **第二步：安装 NodeJS/npm**
如果你没有安装环境，可以通过 Windows 软件包管理器快速安装：
1.  运行安装命令：
    ```bash
    winget install OpenJS.NodeJS.LTS
    ```
2.  **重要：** 安装完成后，必须**关闭当前所有 CMD 窗口并重新打开**，环境配置才会生效。

---

### **第三步：安装 BingoCode**
环境准备就绪后，直接全局安装工具：
```bash
npm install -g bingocode
```

---

### 💡 小贴士
* **权限问题：** 如果在安装第三步时报错，请尝试以 **管理员身份** 运行命令行。
* **验证安装：** 安装完成后，你可以输入 `bingocode`（或该工具定义的启动命令）来确认是否安装成功。
